VA - Vulnerability Assessment

PT - Penetration Testing

Penetration Testing is a process to exploit analyzed vulnerabilities by using automated tools and/or manual excellence. It prioritizes vulnerabilities depending on the criticality levels that are found during the Vulnerability Assessment.ALso measures the level of impact of the vulnerability as well as its worst possible destruction level

VAPT

VAPT is a combination of VA and PT, the purpose of which is to first locate flaws in a system, and then measure vulnerability associated with each of the flaws and categorizes the nature of all possible attacks and raises security alarm before these flaws can be exploited by attackers.

Execution Approach

A security operations center (SOC) is in-houses information security team responsible for monitoring and analyzing an organization’s security on an ongoing basis.

The SOC team’s goal is to detect, analyze, and respond to cybersecurity incidents using a combination of technology solutions and a strong set of processes.

The SOC is responsible for ensuring that potential security incidents are correctly identified, analyzed, defended, investigated, and reported.
